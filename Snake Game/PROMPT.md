Build a complete, standalone retro Snake Game using a clean HTML, CSS, and JavaScript structure. 
Requirements:
1. Mechanics: Classic grid-based movement, continuous speed, score tracker, and a high-score system saved in local storage.
2. Controls: Smooth response using WASD or Arrow Keys. Block the page from scrolling when using arrows.
3. Game Loop: The game ends if the snake hits the wall boundary or runs into its own body. Include a "Game Over" screen with a "Restart" button.
4. Styling: Use a clean, dark cyberpunk theme with a neon green snake and a neon red glowing apple. Center the canvas perfectly on the screen.
5. Code Structure: Put everything into a single index.html file or organize it cleanly into separate index.html, style.css, and script.js files.
